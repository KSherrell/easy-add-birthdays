# easyAdd Birthdays example
## Description
easyAdd Birthdays is a Google Sheets application that was created to help the HR Manager easily manage a birthday reminder calendar. 


## Features
- One-click to run the app
- Adding, removing, and updating birthdays can all happen with that click 
- Birthdays are added to the calendar on a three-year rolling schedule that renews each time the app is run
